# Changelog

## 1.1.0

Initial service-ready release line for the Copilot-first terminal agent.

Highlights:

- common autonomous runtime shared by TUI and `omc run`
- project-aware `/init` and repository context generation
- background sub-agent orchestration with TUI inspection
- context compaction and session recovery tools
- runtime hook system for safer tool use and completion checks
- Copilot custom agent metadata support
- MCP discovery across workspace, editor user config, and global config
- Copilot editor tool visibility through the bridge
- improved TUI message rendering and input stability work
- release verification and deployment checklist
